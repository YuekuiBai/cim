### 第5页：非线性误差建模

**标题**：非线性误差建模

**内容**：

**数学模型**：

```
x' = α · x³ + (1 - α) · x
```

**参数说明**：

| 参数 | 含义              |
| -- | --------------- |
| x  | 理想输入值           |
| x' | 非线性失真后的实际信号     |
| α  | 非线性强度（α>0表示非线性） |

**特性说明**：

- α=0：理想线性关系
- α越大：非线性效应越显著
- α>0时：正输入缩小，负输入放大

**讲解要点**：

- 解释公式的物理含义
- 用图表直观展示不同α下的非线性特性

**配套图表**：可添加非线性函数曲线图

### 第6页：任务一 - 敏感性分析

**标题**：任务一：非线性误差敏感性分析

**内容**：

**实验设置**：

- 模型：ResNet18（CIFAR-10）
- 基准精度：81.95%（α=0）
- 测试α范围：0.0 \~ 0.5

**实验结果**：

| 非线性强度 | 推理精度   | 精度衰减   |
| ----- | ------ | ------ |
| 0.0   | 81.95% | 基准     |
| 0.1   | 80.99% | -1.0%  |
| 0.2   | 77.63% | -4.3%  |
| 0.3   | 70.95% | -11.0% |
| 0.4   | 60.40% | -21.5% |
| 0.5   | 48.49% | -33.5% |

**关键发现**：

- α<0.2：影响可接受（<5%）
- α>0.3：精度急剧下降
- α=0.5：精度损失超30%

**讲解要点**：

- 强调非线性误差的严重性
- 指出需要鲁棒性方法

### 第8页：微调 vs 从头训练

**标题**：微调 vs 从头训练对比

**内容**：

**实验设置**：

- 严谨对比：3次独立实验 × 3个α值
- 计算平均值、标准差、p值

**实验结果**：

| 训练策略 | α=0.1  | α=0.2  | α=0.3  | 平均         | Std    |
| ---- | ------ | ------ | ------ | ---------- | ------ |
| 微调   | 85.25% | 85.40% | 84.98% | **85.21%** | ±0.11% |
| 从头训练 | 81.40% | 81.25% | 81.30% | 81.32%     | ±0.13% |

**差异**：**+3.85% \~ +4.15%**，p-value < 0.0001

**关键发现**：

- 微调平均精度更高
- 波动更小
- 收敛更快
- 统计显著性高度显著

**讲解要点**：

- 强调微调策略的优势

### 第10页：高级方法探索

**标题**：高级方法探索

**内容**：

**NAT+混合Alpha**：

- 每次迭代随机采样不同α值
- 模型同时适应多种噪声水平
- 全范围均衡，平均81.71%

**课程NAT**：

- 借鉴课程学习思想
- 从低噪声到高噪声渐进训练
- 高α区域保护效果好（82.48%）

**OVF训练**：

- 基于负反馈理论
- 前向传播注入定向噪声
- 平均81.06%，波动6.24%

**综合对比**：

| 方法              | 平均精度       | 波动        | 特点      |
| --------------- | ---------- | --------- | ------- |
| NAT固定           | 78.25%     | 30.98%    | 峰值高但波动大 |
| **NAT+混合Alpha** | **81.71%** | **5.53%** | 最佳均衡    |
| 课程NAT           | 79.47%     | 13.18%    | 高α保护好   |
| OVF训练           | 81.06%     | 6.24%     | 理论支撑    |

**讲解要点**：

- 介绍创新方法的原理
- 强调NAT+混合Alpha的最优性
