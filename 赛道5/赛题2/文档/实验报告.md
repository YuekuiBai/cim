## 1. 任务一：通用STE框架设计与实现

### 1.1 框架设计目标

根据赛题二要求，设计能够处理存算芯片复杂噪声特性的直通估计器算法：

| 误差类型 | 处理要求 |
| --- | --- |
| **编程误差的加性噪声** | 考虑噪声的均值偏移和方差变化 |
| **非线性噪声的乘性特征** | 设计针对乘性噪声的梯度估计策略 |
| **输出误差的量化效应** | 处理量化操作的梯度传播 |

### 1.2 STE框架组成

| 组件 | 文件 | 说明 |
| --- | --- | --- |
| **NoiseInjector** | ste/core.py | 噪声注入器，模拟存算芯片各类噪声 |
| **NoisyLinear** | ste/core.py | 带噪声的线性层 |
| **NoisyConv2d** | ste/core.py | 带噪声的卷积层 |
| **STEGradientEstimator** | ste/core.py | 梯度估计器 |
| **AdaptiveSTE** | ste/innovation.py | 自适应STE梯度估计器 |
| **NoiseAwareRegularizer** | ste/innovation.py | 噪声感知正则化器 |
| **BiasCorrector** | ste/innovation.py | 偏差校正机制 |
| **LayerwiseNoiseInjection** | ste/innovation.py | 层次化噪声注入 |

### 1.3 梯度估计核心算法

我们的STE框架采用以下梯度估计策略：

```python
class STEGradientEstimator:
    def estimate(self, grad_output, input=None):
        # 恒等梯度：∂L/∂x ≈ ∂L/∂y
        grad_input = grad_output

        # 可选：自适应缩放
        if self.adaptive_scale:
            grad_input = grad_input * self._get_scale_factor()

        return grad_input
```

## 5. 创新性改进方案

### 5.1 理论基础创新

#### 5.1.1 梯度方向保持性证明

参考CoRR 2025论文，我们提出以下理论框架：

**命题1（梯度方向保持）：**
在STE框架下，反向传播的梯度方向与真实梯度方向满足：
$$\cos(\theta) = \frac{\nabla_w L \cdot \tilde{\nabla}_w L}{\|\nabla_w L\| \|\tilde{\nabla}_w L\|} \geq \rho$$

其中$\rho$为方向保持系数，在适当的噪声水平下可保证$\rho > 0$。

#### 5.1.2 收敛性分析

**命题2（收敛性条件）：**
假设损失函数$L$满足L-smooth条件，STE梯度$\tilde{g}$满足：
$$\mathbb{E}[\tilde{g}] = \nabla L + b(w), \quad \mathbb{E}[\|\tilde{g} - \mathbb{E}[\tilde{g}]\|^2] \leq \sigma^2$$

则在适当的步长选择下，STE优化可收敛到损失面的稳定邻域。

### 5.2 算法创新

#### 5.2.1 自适应STE（Adaptive STE）

```python
class AdaptiveSTE(nn.Module):
    def __init__(self, base_estimator='identity'):
        self.base_estimator = base_estimator
        self.noise_strength = 1.0
        self.adaptive_scale = True
        self.noise_schedule = 'sqrt'  # 四种调度可选

    def estimate(self, grad_output, input=None, noise_level=0.0):
        base_grad = self._base_ste(grad_output)

        if self.adaptive_scale:
            scale = self._get_adaptive_scale(noise_level)
            base_grad = base_grad * scale

        return base_grad

    def _get_adaptive_scale(self, noise_level):
        if self.noise_schedule == 'sqrt':
            return 1.0 / np.sqrt(1.0 + noise_level ** 2)
        elif self.noise_schedule == 'inverse':
            return 1.0 / (1.0 + noise_level ** 2)
        elif self.noise_schedule == 'linear':
            return 1.0 / (1.0 + noise_level)
        elif self.noise_schedule == 'exp':
            return np.exp(-noise_level / 2.0)
        return 1.0
```

**核心创新**：四种噪声调度策略，根据噪声水平动态调整梯度缩放因子

#### 5.2.2 噪声感知正则化

```python
class NoiseAwareRegularizer:
    def __init__(self, sigma=0.01):
        self.sigma = sigma

    def compute_penalty(self, model):
        penalty = 0.0
        for name, param in model.named_parameters():
            if 'weight' in name:
                penalty = penalty + torch.sum(param ** 2)
        return self.sigma * penalty
```

**注**：实验发现正则化系数需要进一步调优，当前设置(σ=0.01)导致训练不稳定

#### 5.2.3 偏差校正机制

```python
class BiasCorrector:
    def __init__(self, correction_type='ema'):
        self.correction_type = correction_type
        self.ema_bias = None
        self.ema_decay = 0.9

    def correct(self, grad, noise_config, noise_strength):
        prog_noise_std = noise_config.get('prog_noise_std', 0.01) * noise_strength
        drift_factor = noise_config.get('drift_factor', 0.005) * noise_strength
        batch_bias = -grad * (prog_noise_std ** 2 + drift_factor ** 2)

        if self.ema_bias is None:
            self.ema_bias = batch_bias
        else:
            self.ema_bias = self.ema_decay * self.ema_bias + (1 - self.ema_decay) * batch_bias

        return grad - self.ema_bias
```

### 5.3 架构创新

#### 5.3.1 异构噪声处理

| 噪声类型 | 处理策略 | 梯度估计方法 |
| --- | --- | --- |
| 加性噪声 | 方差稳定化 | 缩放校正 |
| 乘性噪声 | 对数空间处理 | 相对梯度 |
| 量化噪声 | 均匀近似 | STE恒等 |

#### 5.3.2 层次化噪声注入

```python
class LayerwiseNoiseInjection:
    def __init__(self, noise_config):
        self.noise_config = noise_config
        self.layer_scales = {
            'conv1': 0.8,
            'conv2': 1.0,
            'fc': 1.2,
        }

    def get_layer_noise(self, layer_name, layer_depth, layer_type):
        base_scale = self.layer_scales.get(layer_type, 1.0)
        depth_factor = 1.0 / (1.0 + 0.1 * layer_depth)
        return base_scale * depth_factor
```
