## 1. 概述

### 1.1 框架简介

本框架实现了基于直通估计器（STE）的噪声感知训练方法，专门用于存算一体（CIM）芯片上的神经网络训练。框架支持：

| 功能 | 说明 |
| --- | --- |
| **多类型噪声注入** | 编程误差、非线性、输出噪声 |
| **自适应STE** | 四种调度策略（Inverse/Linear/Sqrt/Exp） |
| **噪声感知正则化** | 降低模型对噪声的敏感性 |
| **偏差校正机制** | EMA方法补偿梯度偏差 |
| **层次化噪声注入** | 根据层类型差异化处理 |

## 4. 核心功能

### 4.1 噪声注入器

```python
from ste.core import NoiseInjector

noise_config = {
    'prog_noise_std': 0.01,      # 编程噪声标准差
    'drift_factor': 0.005,        # 权重漂移因子
    'nonlinear_alpha': 0.1,       # 正向饱和参数
    'nonlinear_beta': 0.05,       # 负向饱和参数
    'output_noise_std': 0.01,     # 输出噪声标准差
    'crosstalk_factor': 0.002     # 串扰因子
}

injector = NoiseInjector(noise_config)
noisy_output = injector.inject_to_output(output)
```

### 4.2 自适应STE

```python
from ste.innovation import AdaptiveSTE

adaptive_ste = AdaptiveSTE(
    estimator_type='identity',
    adaptive_scale=True,
    noise_schedule='sqrt'  # 四种策略可选
)

adaptive_ste.set_noise_level(1.0)
estimated_grad = adaptive_ste.estimate(grad_output)
```

### 4.3 噪声感知正则化

```python
from ste.innovation import NoiseAwareRegularizer

regularizer = NoiseAwareRegularizer(
    sigma=0.01,
    lambda_reg=0.001
)

penalty = regularizer.compute_penalty(model, noise_config)
loss = main_loss + penalty
```

### 4.4 偏差校正

```python
from ste.innovation import BiasCorrector

corrector = BiasCorrector(
    correction_type='ema',
    ema_decay=0.9
)

corrected_grad = corrector.correct(grad, noise_config, noise_strength)
```

### 4.5 层次化噪声注入

```python
from ste.innovation import LayerwiseNoiseInjection

layerwise = LayerwiseNoiseInjection(noise_config)

noise_scale = layerwise.get_layer_noise_scale(
    layer_name='layer1.conv1',
    layer_depth=1,
    layer_type='conv2'
)
```

## 5. 参数配置

### 5.1 噪声参数

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| prog_noise_std | 0.01 | 编程噪声标准差，相邻器件间可能有空间相关性 |
| drift_factor | 0.005 | 权重随时间的漂移因子 |
| nonlinear_alpha | 0.1 | 正向饱和非线性参数 |
| nonlinear_beta | 0.05 | 负向饱和非线性参数 |
| output_noise_std | 0.01 | ADC输出噪声标准差 |
| crosstalk_factor | 0.002 | 器件间串扰因子 |

### 5.2 训练参数

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| epochs | 30 | 训练轮次 |
| lr | 0.1 | 初始学习率 |
| momentum | 0.9 | SGD动量 |
| weight_decay | 5e-4 | 权重衰减 |
| batch_size | 128 | 批大小 |

## 7. 常见问题

### Q1: 训练过程中精度停滞怎么办？

**可能原因：**
- 学习率过大或过小
- 噪声强度过高
- 正则化系数过大

**解决方案：**
```python
# 调整学习率
optimizer = torch.optim.SGD(model.parameters(), lr=0.05, momentum=0.9)

# 降低噪声强度
noise_strength = 0.5

# 减小正则化系数
lambda_reg = 0.001
```

### Q2: GPU内存不足怎么办？

**解决方案：**
```bash
# 减小批大小
python run_ste.py --batch_size 64

# 启用梯度累积
# 在代码中设置 accumulation_steps = 4
```

### Q3: 如何选择合适的噪声调度策略？

**推荐：**
| 场景 | 推荐策略 |
| --- | --- |
| 高噪声环境 | Sqrt |
| 中等噪声 | Linear |
| 低噪声 | Inverse |
| 极低噪声 | Exp |

**实验结果：** Sqrt策略在大多数场景下表现最佳。

### Q4: 正则化导致训练不稳定？

**原因：** lambda_reg过大

**解决方案：**
```python
# 减小正则化系数
regularizer = NoiseAwareRegularizer(
    sigma=0.01,
    lambda_reg=0.0001  # 从0.001降到0.0001
)
```

### Q5: 如何复现实验结果？

```python
# 设置随机种子
torch.manual_seed(42)
torch.cuda.manual_seed(42)
torch.backends.cudnn.deterministic = True
torch.backends.cudnn.benchmark = False
```

## 附录：完整示例代码

```python
import torch
import torch.nn as nn
import torch.optim as optim
from ste.core import inject_ste_to_model
from ste.innovation import AdaptiveSTE, BiasCorrector

noise_config = {
    'prog_noise_std': 0.01,
    'drift_factor': 0.005,
    'nonlinear_alpha': 0.1,
    'nonlinear_beta': 0.05,
    'output_noise_std': 0.01,
    'crosstalk_factor': 0.002
}

model = nn.Sequential(
    nn.Conv2d(3, 64, 3, padding=1),
    nn.ReLU(),
    nn.Conv2d(64, 128, 3, padding=1),
    nn.ReLU(),
    nn.AdaptiveAvgPool2d(1),
    nn.Flatten(),
    nn.Linear(128, 10)
)

model = inject_ste_to_model(model, noise_config)
optimizer = optim.SGD(model.parameters(), lr=0.1, momentum=0.9)
adaptive_ste = AdaptiveSTE(noise_schedule='sqrt')
bias_corrector = BiasCorrector()

for epoch in range(30):
    for data, target in dataloader:
        optimizer.zero_grad()

        output = model(data)
        loss = nn.functional.cross_entropy(output, target)
        loss.backward()

        for param in model.parameters():
            if param.grad is not None:
                param.grad = adaptive_ste.estimate(param.grad)
                param.grad = bias_corrector.correct(
                    param.grad, noise_config, 1.0
                )

        optimizer.step()

    print(f"Epoch {epoch+1}/30 completed")
```
