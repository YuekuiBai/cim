## 共同技术背景

### 存算一体芯片概述

存算一体（In-Memory Computing, IMC）芯片通过将存储和计算功能深度融合，从根本上解决了传统冯·诺依曼架构中数据频繁搬运造成的功耗和延迟问题。这种架构特别适合深度学习应用，因为神经网络推理过程中大量的矩阵向量乘法运算可以直接在存储阵列中以模拟计算的方式高效完成。

### 主要误差来源

| 误差类型 | 物理来源 | 影响 |
|---------|---------|------|
| **非线性误差** | 器件 I-V 特性非线性、跨导变化 | 计算结果偏离理想值 |
| **编程误差** | 权重写入过程中编程脉冲精度限制 | 与权重幅值相关的加性噪声 |
| **器件噪声** | 热噪声、散粒噪声、1/f 噪声 | 输出信号随机波动 |
| **器件变异** | 制造工艺波动、材料不均匀 | 单元间响应不一致 |
| **量化误差** | ADC/DAC 有限精度 | 信息丢失 |
| **时序误差** | 信号传播延迟、建立时间不足 | 动态误差 |

### 算法优化的必要性

硬件层面的改进（如工艺优化、电路设计）成本高、周期长，而**算法层面的优化**可以：

- 快速迭代验证
- 成本低、灵活性强
- 与硬件方案协同优化
- 适用于已部署系统的升级

## 赛题一：非线性误差对推理精度的影响研究

### 赛题任务

本研究面向存算一体芯片部署场景，针对模拟域乘累加运算中固有的输入相关非线性失真问题，将非线性映射模型嵌入到可进行矩阵计算的算子（包括 linear，conv2d，convtranspose2d 等）的输入（激活值）中，对激活值非线性误差进行如下分析与研究：

#### 任务 1：非线性误差的敏感性分析

在特定任务中，系统评估不同非线性强度（α 取不同值）对网络推理精度的影响，包括但不限于：

- 整网精度衰减趋势
- 单层输出分布偏移
- 误差在网络中的逐层累积行为

#### 任务 2：非线性感知训练（Nonlinearity-Aware Training）

在训练阶段加入非线性映射模型，探究其对网络收敛性、泛化性的影响；对比微调（fine-tuning）与从头训练（training from scratch）策略对非线性鲁棒性的影响。

#### 任务 3：鲁棒性增强方法设计

针对非线性误差，提出特定的优化方法，提升模型对抗非线性误差的能力。

#### 拓展研究

- 网络结构与参数量对非线性误差影响
- 对比分析随机高斯噪声注入与非线性失真对模型鲁棒性的不同影响机制
- 结合量化误差与非线性误差对模型推理精度的影响进行分析

### 非线性建模

**非线性映射函数：**

```
x' = α · x³ + (1 - α) · x
```

| 参数 | 含义 |
|------|------|
| x | 经归一化处理后的理想输入值 |
| x' | 经非线性失真后的实际输入信号 |
| α | 非线性程度参数（α > 0 表示非线性强度） |

**非线性特性说明：**

| 条件 | 特性 |
|------|------|
| α = 0 | 映射退化为理想线性关系 |
| α 越大 | 器件非线性效应越显著 |
| α > 0 时 | 正输入区域 (x > 0) 呈现缩小效应，负输入区域 (x < 0) 呈现放大效应 |
| α < 0 时 | 效果相反 |

**非线性误差使用实例：**

```python
def nonlinearity(x, alpha=0.0):
    max_val = x.abs().max()
    x = x / max_val
    y = alpha * (x**3) + (1 - alpha) * x
    return y * max_val
```

## 技术详解

### 非线性建模方法

#### 物理来源分析

| 非线性来源 | 描述 | 典型模型 |
|-----------|------|---------|
| **MOSFET 亚阈值导通** | 栅源电压低于阈值时的指数特性 | $I \propto e^{V/V_T}$ |
| **迁移率退化** | 高电场下载流子迁移率下降 | $I \propto V - \alpha V^2$ |
| **接触电阻** | 金属-半导体接触非线性 | $V = IR + \beta I^2$ |
| **饱和效应** | 电流饱和限制 | $I = I_{sat} \cdot \tanh(V/V_0)$ |

#### 误差补偿方法

| 方法 | 实现难度 | 补偿效果 | 计算开销 |
|------|---------|---------|---------|
| **逆非线性预失真** | ⭐⭐ | ⭐⭐⭐ | ⭐ |
| **校准层** | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ |
| **非线性感知微调** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

### 噪声感知训练方法

#### 噪声模型

| 噪声类型 | 分布 | 公式 |
|---------|------|------|
| **高斯噪声** | $\mathcal{N}(0, \sigma^2)$ | $y = x + \epsilon$ |
| **乘性噪声** | 对数正态 | $y = x \cdot (1 + \epsilon)$ |
| **量化噪声** | 均匀分布 | $y = \text{round}(x/\Delta) \cdot \Delta$ |
| **脉冲噪声** | 伯努利 | $y = x \cdot m$ |

#### 直通估计器（STE）

**前向传播：** y = f(x)，其中 f 为不可微函数

**反向传播：** ∂L/∂x = ∂L/∂y · ∂y/∂x ≈ ∂L/∂y · I

## 常见问题解答

### Q1: 从什么模型开始比较好？

**建议从 ResNet-18 + CIFAR-10 开始：**
- 模型结构简单清晰
- 数据集小，训练快
- 容易调试和验证
- 结果有参考基准

### Q2: 如何确定合适的噪声/非线性强度？

**参考方法：**
1. 查阅文献中报道的实验测量值
2. 从较小噪声开始（1%-5%）
3. 逐步增加，观察精度下降趋势
4. 选择精度下降 10%-20% 的强度作为目标

### Q3: 噪声感知训练影响清洁精度怎么办？

**可能原因与解决：**
- **噪声过强** → 降低噪声强度或采用渐进式训练
- **训练不足** → 增加训练轮数
- **学习率不当** → 调整学习率或使用学习率调度
- **多噪声条件训练** → 提高泛化能力

### Q4: 如何证明方法的有效性？

**对比实验设计：**
1. **基线模型** - 无噪声训练
2. **你的方法** - 噪声感知训练
3. **对比方法** - 其他鲁棒训练方法
4. **多噪声水平测试** - 验证泛化能力
5. **统计显著性** - 多次实验取平均
