## 一、两阶段闭环科研体系

### 1.1 体系架构

```
第一阶段：专利创新验证（局部实验）                    第二阶段：SOTA冲刺（整体实验）
┌─────────────────────────────────────┐              ┌─────────────────────────────────────┐
│ 专利创新点提取                      │              │ 多专利创新点组合                    │
│  ↓                                  │              │  ↓                                   │
│ 局部消融实验（验证单一创新有效性）  │              │ 对比实验（与SOTA方法对比）          │
│  ↓                                  │              │  ↓                                   │
│ 实验结果分析                        │              │ 精度冲击90%+                        │
│  ↓                                  │              │  ↓                                   │
│ ┌─────────────────────┐            │              │ ┌─────────────────────┐             │
│ │ 效果好 → 专利数据更新 │────────────┼──────────────┤│ 效果好 → 专利组合    │             │
│ │ 效果差 → 专利创新改进 │            │              │ │ 效果差 → 新方法探索  │             │
│ └─────────────────────┘            │              │ └─────────────────────┘             │
│  ↓                                  │              │  ↓                                   │
│ 专利交底书同步更新                  │              │ 技术报告撰写                        │
└─────────────────────────────────────┘              └─────────────────────────────────────┘
```

### 1.2 核心原则

1. **每篇专利必须有实验支撑**：创新点→消融实验→对比实验→专利更新
2. **局部与整体并重**：单点验证 + 组合优化缺一不可
3. **失败也是成果**：实验效果差同样有价值，需分析原因并改进专利
4. **数据驱动专利迭代**：专利交底书必须包含最新实验数据

## 二、专利架构调整说明

### 2.1 赛题1专利精简（三篇→两篇）

**原架构**：
- 专利一：混合Alpha随机采样
- 专利二：逐层诊断与补偿（独立专利，但补偿效果仅60.74%）
- 专利三：课程学习+SAM+OVF

**新架构**：
- **专利一**：混合Alpha随机采样 + 分层Alpha训练策略（含原专利二的诊断框架）
  - 核心：随机α采样 + 逐层敏感性诊断 + 差异化α范围
  - 诊断框架不再独立成篇，而是作为专利一的从属权利要求（权利要求7-8、10）
  - 实验目标：87.33% → 89~90%+
- **专利二**：课程学习渐进式非线性鲁棒训练（原专利三）
  - 核心：8阶段渐进α + SAM + OVF变分融合
  - 实验目标：82.48% → 85%+

**精简理由**：
1. 原专利二单独验证仅60.74%，作为独立专利价值有限
2. 诊断框架的核心价值在于指导专利一的分层训练，而非独立补偿
3. 精简后赛题1保留两篇高质量专利：一篇面向中等噪声（专利一），一篇面向极端噪声（专利二）

### 2.2 赛题2专利架构（三篇不变）

- **专利四**：自适应STE四种调度策略（Inverse/Linear/Sqrt/Exp）
- **专利五**：层次化噪声注入框架（AR(1)+γ_l公式）
- **专利六**：偏差校正与正则化（EMA+噪声感知正则化）

## 三、五篇专利逐个攻克计划

### 3.1 赛题1 - 两篇专利

#### 专利一：混合Alpha随机采样 + 分层Alpha训练

| 维度 | 内容 |
|------|------|
| **核心创新** | 随机α采样 + 逐层敏感性诊断 → 差异化α范围（高敏感U(0,0.15)、中敏感U(0,0.3)、低敏感U(0,0.5)） |
| **当前验证** | 87.33%（ResNet18, 20 epochs，无诊断） |
| **消融实验** | 诊断模块消融、分层α范围消融、敏感性阈值消融 |
| **对比实验** | vs全局统一α、vs固定α训练、vs专利二课程学习 |
| **实验脚本** | `run_patent1_2_combined.py` |
| **GPU分配** | GPU2（6-8小时） |
| **优先级** | P0（最高） |
| **预期精度** | 89~90%+ |

**实验配置：**
```
配置1：ResNet34 + 诊断+分层α(标准阈值) + Mixup + EMA + 100epochs
配置2：ResNet34 + 诊断+分层α(严格阈值) + Mixup + EMA + 100epochs
配置3：ResNet34 + 诊断+分层α(宽松阈值) + Mixup + EMA + 100epochs
配置4：消融-无诊断（全局α）→ 验证诊断模块贡献
```

#### 专利二（原专利三）：课程学习+SAM+OVF

| 维度 | 内容 |
|------|------|
| **核心创新** | 8阶段渐进α(0.05→0.4) + SAM平坦最优解 + OVF变分融合 |
| **当前验证** | 82.48%@α=0.5（SAM/OVF未充分消融） |
| **消融实验** | SAM独立贡献、OVF独立贡献、阶段数消融(4/8/12) |
| **对比实验** | vs直接高噪声训练、vs专利一混合训练 |
| **GPU分配** | GPU1（4-6小时） |
| **优先级** | P1 |
| **预期精度** | 85%+（极端噪声α=0.5） |

**实验配置：**
```
配置1：仅课程学习 → 验证基线82.48%
配置2：课程学习+SAM → 验证SAM贡献
配置3：课程学习+OVF → 验证OVF贡献
```
