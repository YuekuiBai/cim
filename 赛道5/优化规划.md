## 二、SOTA文献调研

### 2.1 核心参考论文

#### 【最相关】Extended STE for Analog CIM (arXiv:2508.11940, 2025.08)
- **标题**：Extending Straight-Through Estimation for Robust Neural Networks on Analog CIM Hardware
- **核心思想**：将前向噪声仿真与反向梯度计算解耦，实现更精确但计算复杂的噪声建模
- **性能提升**：图像分类精度提升5.3%，训练速度提升2.2倍，峰值显存降低37.9%
- **理论贡献**：保持梯度方向信息，保证优化稳定性
- **启发点**：赛题2的STE框架应借鉴前向/反向解耦思想

#### Column-wise Quantization (arXiv:2502.07842, 2025.03)
- **核心思想**：列级权重和部分和量化对齐，提高ADC分辨率利用率
- **性能提升**：ResNet-20(CIFAR-10)精度+0.99%，ResNet-20(CIFAR-100)+2.69%，ResNet-18(ImageNet)+1.01%
- **启发点**：联合优化权重和部分和量化粒度

#### 南京大学高精度CIM芯片 (Science Advances, 2025)
- **核心思想**：将权重实现从器件物理参数转换为器件几何尺寸比
- **性能**：0.101% RMS误差，极端温度(-78.5°C~180°C)下误差<0.155%
- **启发点**：权重重映射技术的理论依据

#### ISSCC STT-MRAM CIM (2025)
- **核心思想**：μN-ΔI异构CIM架构 + 贝叶斯神经网络
- **性能**：噪声环境下BNN精度退化比CNN低2.55倍
- **启发点**：噪声鲁棒性架构设计

### 2.2 技术路线对比

| 方法 | 核心思想 | 优点 | 缺点 |
|:-----|:---------|:-----|:-----|
| 赛题2现有方法 | STE+正则化 | 简单直接 | 效果差(55.86%) |
| Extended STE | 前向/反向解耦 | 5.3%精度提升 | 需复杂噪声建模 |
| Column Quant | 列级量化对齐 | ImageNet验证 | 硬件改动大 |
| BNN+CIM | 概率推断 | 噪声鲁棒 | 实现复杂 |

## 四、具体改进方案

### 4.1 赛题二改进（优先级：高）

#### Step 1：借鉴Extended STE核心思想（预计+2分）
**论文核心**：前向噪声仿真与反向梯度计算解耦

**改进实现**：
```python
class DecoupledSTE(nn.Module):
    """
    前向-反向解耦的噪声感知训练
    核心创新：前向使用真实噪声，反向使用修正STE
    """
    def forward(self, x, noise_strength):
        # 前向：注入真实CIM噪声
        noise = self.sample_cim_noise(x, noise_strength)
        x_noisy = x + noise
        return self.linear(x_noisy)

    def ste_backward(self, grad):
        # 反向：使用扩展STE，保持梯度方向
        return grad * self.get_correction_factor()
```

**预期效果**：精度从55.86%提升至接近基准85%

#### Step 2：实现噪声敏感度自适应（预计+1分）
- 自动识别不同层对噪声的敏感度
- 对敏感层使用更强的正则化或更小的噪声
- 借鉴贝叶斯神经网络思想进行不确定性量化

#### Step 3：目标检测/语义分割验证（加分项，预计+3分）
- 使用YOLO进行COCO目标检测验证
- 使用UNet进行Cityscapes语义分割验证
- **备选**：如数据集下载过慢，使用CIFAR-100代替
