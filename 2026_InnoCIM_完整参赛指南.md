## 赛道二：编译工具链

### 一句话看懂

实现从 **ONNX 模型** 到 **存算一体硬件指令** 的完整编译流程。

### 任务

统一编译任务：将 ONNX 模型（linear、elementwise 算子）转换为硬件 ISA 指令。

### 硬件规格

| 组件       | 规格                 |
| -------- | ------------------ |
| **存算单元** | 1024bit × 4096bit  |
| **SRAM** | 512KB              |
| **数据类型** | int8 / int16 / i32 |

### 基础指令集

| 指令                         | 功能                              |
| -------------------------- | ------------------------------- |
| `cim.bit.type`             | 输入 bit 级存内计算                    |
| `elt.op.type.mode`         | Elementwise 运算（add/sub/mul/div） |
| `mem.copy.dstType.srcType` | 数据拷贝                            |

### 评分标准

| 模块                 |  分值 |
| ------------------ | :-: |
| IR设计（表达能力、简洁性、扩展性） |  20 |
| 模型解析（ONNX → IR）    |  10 |
| 指令生成               |  15 |
| 资源管理策略             |  15 |
| Transform/Lowering |  20 |
| 技术路线               |  10 |
| 设计报告               |  5  |
| 现场答辩               |  5  |

### 入门标准

- ✅ 正确解析 ONNX 模型
- ✅ 生成符合规范的指令序列
- ✅ SRAM 分配合理，无越界
