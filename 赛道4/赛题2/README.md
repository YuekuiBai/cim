# 赛道四 - 赛题二：基于 FET 类器件的模拟存内计算设计

## 项目概述

本项目针对赛题二要求，以 FET 类存储器件为核心，完成从器件选型、工作区间分析到模拟存内计算阵列设计的全流程方案。重点解决多比特正负输入/权重表达、输出线性度优化及计算精度评估。

## 技术方案

| 设计环节 | 方案 |
|---------|------|
| 器件选型 | FeFET（铁电场效应晶体管） |
| 工作区间 | 亚阈值区 + 线性区组合，兼顾多比特精度与线性度 |
| 正负表达 | 差分对管结构（正负权重/输入分离） |
| 线性度补偿 | 预失真校准 + 分段线性拟合 |
| 精度评估 | 温度特性(±20°C)、噪声分析、器件失配(σ=3%) |

## 核心指标

| 指标 | 数值 |
|------|------|
| 输入精度 | 4-bit (±8级) |
| 权重精度 | 4-bit (±8级) |
| 输出精度 | 8-bit |
| 线性度 | R² > 0.99（补偿后） |
| 温度范围 | -20°C ~ 80°C |
| 失配容忍 | σ < 5% |
