# 第一章 FeFET器件的工作原理及特性介绍

## 1.1 FeFET器件简介

FeFET（铁电场效应晶体管）是一种结合了铁电材料与MOSFET结构的存储器件。其核心特点是在栅极介电层中引入铁电材料，利用铁电极化状态来调制沟道电导，实现非易失性存储功能。

### 器件结构

FeFET的基本结构为MFS（Metal-Ferroelectric-Semiconductor）结构：

| 层级 | 材料 | 功能 |
|------|------|------|
| 栅极 | 金属 | 控制电极 |
| 铁电层 | HfO2/PZT/SBT | 极化调制 |
| 氧化层 | SiO2 | 界面隔离 |
| 沟道 | Si | 导电通路 |

### 与传统晶体管的区别

- 传统MOSFET：栅极电压决定沟道电导（易失性）
- FeFET：铁电极化决定沟道电导（非易失性）
- 极化状态可保持数年而不需供电

## 1.2 工作原理

### 铁电极化机制

FeFET利用铁电材料的双稳态极化特性：

**极化状态**：
- P+（正向极化）：低阈值电压
- P-（反向极化）：高阈值电压

### 阈值电压调制

```
Vth = Vth0 + γ·P
```
其中：
- Vth0：初始阈值电压
- γ：铁电耦合系数
- P：极化强度

### SET/RESET操作

| 操作 | 施加电压 | 极化状态 | 阈值电压 |
|------|---------|----------|----------|
| SET | +Vpp | P+ | Vth_L |
| RESET | -Vpp | P- | Vth_H |

## 1.3 电学特性

### I-V特性

FeFET的漏电流表达式（平方率模型）：

**线性区 (Vds < Vgs - Vth)**:
```
Ids = μ·Cox·(W/L)·[(Vgs - Vth)·Vds - Vds²/2]
```

**饱和区 (Vds ≥ Vgs - Vth)**:
```
Ids = (1/2)·μ·Cox·(W/L)·(Vgs - Vth)²
```

### 关键参数

| 参数 | 典型值 | 单位 |
|------|-------|------|
| 阈值电压范围 | 0.3 - 1.2 | V |
| 亚阈值摆幅 | 60 - 70 | mV/decade |
| 漏电流(关态) | < 1 | pA/μm |
| 极化保持时间 | > 10 | 年 |
| 写入 endurance | > 10^10 | cycles |

## 1.4 多比特存储机制

FeFET可通过调节极化程度实现多比特存储：

### 多比特写入

通过对铁电层施加不同幅度和宽度的脉冲，可以精确控制极化状态：

```
量化等级数 = 2^n (n为比特数)
```

### 阈值电压分布

| 状态 | 阈值电压 | 存储值 |
|------|----------|--------|
| 00 | 0.3 - 0.5V | 00 |
| 01 | 0.5 - 0.7V | 01 |
| 10 | 0.7 - 0.9V | 10 |
| 11 | 0.9 - 1.2V | 11 |

## 1.5 存内计算实现原理

### 模拟计算原理

在存内计算应用中，FeFET的沟道电流与输入电压和权重的乘积成正比：

```
Iout = g(Vin) · W
```

其中：
- Vin：输入电压（模拟输入信号）
- W：权重量化值（由阈值电压决定）
- g()：跨导函数

### 矩阵-向量乘法

利用交叉阵列结构，可以高效实现矩阵-向量乘法：

```
[Output] = [Input] × [Weight Matrix]
```
