# 第三章 正负信号处理方案

## 3.1 正负信号的挑战

在神经网络计算中，输入、权重和输出都可能为正或为负。模拟存内计算需要有效处理这些正负信号。

### 问题描述

| 信号类型 | 数学表示 | 模拟实现 |
|----------|----------|----------|
| 正输入 | X > 0 | 正电压 |
| 负输入 | X < 0 | 负电压或偏移 |
| 正权重 | W > 0 | Vth降低 |
| 负权重 | W < 0 | Vth升高 |
| 正输出 | Y > 0 | 正电流 |
| 负输出 | Y < 0 | 差分电流 |

### 传统方案对比

| 方案 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 双电源 | 正负电源 | 直观 | 电路复杂 |
| 电荷域 | 电荷运算 | 线性好 | 工艺敏感 |
| 电流域 | 差分电流 | 适合CIM | 噪声叠加 |

## 3.2 差分对电路方案

### 电路结构

采用差分对实现正负权重的统一表达：

```
权重范围: [-1, +1] → [Vth_L, Vth_H]
```

### 工作原理

**正权重处理**：
- W > 0：Vth向低值偏移
- Iout = g(Vin) · W

**负权重处理**：
- W < 0：Vth向高值偏移
- Iout = -g(|Vin|) · |W|

### 差分电流产生

通过两个晶体管分别处理正负分量：

```
Iout = I_pos - I_neg
```

其中：
- I_pos = g(Vin_pos) · W_pos
- I_neg = g(Vin_neg) · W_neg

## 3.3 多比特正负输入处理

### 编码方案

| 输入值范围 | 编码方式 | 表示 |
|------------|----------|------|
| [-1, 0) | 偏移二进制 | 0x0000 - 0x7FFF |
| [0, 1] | 偏移二进制 | 0x8000 - 0xFFFF |

### 4-bit输入编码示例

| 输入值 | 量化码 | 模拟电压 |
|--------|--------|----------|
| -1.0 | 0000 | 0.0V |
| -0.5 | 0100 | 0.45V |
| 0.0 | 1000 | 0.9V |
| +0.5 | 1100 | 1.35V |
| +1.0 | 1111 | 1.8V |

## 3.4 多比特正负权重处理

### 权重编码

权重值范围 [-1, 1] 通过阈值电压调制实现：

| 权重值 | 阈值电压 | 物理状态 |
|--------|----------|----------|
| -1.0 | Vth_H (1.2V) | 高阻态 |
| -0.5 | Vth_H - Δ | 中高阻 |
| 0.0 | Vth_center (0.75V) | 中性 |
| +0.5 | Vth_L + Δ | 中低阻 |
| +1.0 | Vth_L (0.3V) | 低阻态 |

### 权重映射表

| 量化码(6-bit) | 权重值 | Vth (V) | 状态 |
|---------------|--------|---------|------|
| 000000 | -1.000 | 1.200 | P- |
| 001000 | -0.500 | 0.975 | P- |
| 010000 | 0.000 | 0.750 | Neutral |
| 011000 | +0.500 | 0.525 | P+ |
| 100000 | +1.000 | 0.300 | P+ |

## 3.5 差分阵列设计

### 128×128差分阵列

```
行: 128 (输入)
列: 128 (输出)

差分结构:
  - 正权重列: W > 0
  - 负权重列: W < 0
  - 输出: Iout = I_pos - I_neg
```

### 路由方案

```
输入电压 → DAC → 字线驱动器 → FeFET阵列 → 位线累加 → ADC
                ↓
           参考电压(Vref)
```

## 3.6 输出电流处理

### 电流-电压转换

差分电流转换为电压：

```
Vout = Iout · R_load
```

### 双极性输出

| 输出范围 | I > 0 | I < 0 |
|----------|-------|-------|
| Y > 0 | 正电流 | - |
| Y < 0 | - | 负电流 |
| Y = 0 | I_pos = I_neg | - |
