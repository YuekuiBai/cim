# 第四章 线性度分析与补偿

## 4.1 线性度的重要性

线性度是衡量模拟存内计算系统精度的关键指标。理想情况下，计算结果应满足：

```
Y = X · W
```

其中Y与X和W呈线性关系。

### 线性度指标

| 指标 | 定义 | 目标值 |
|------|------|--------|
| 相关系数R² | 实际vs理想输出的相关性 | > 0.99 |
| 最大误差 | \|Y_actual - Y_ideal\| | < 1% FSR |
| 微分非线性(DNL) | 码间宽度变化 | < 1 LSB |
| 积分非线性(INL) | 传递函数偏移 | < 2 LSB |

## 4.2 FeFET非线性来源

### 器件固有非线性

| 非线性来源 | 机制 | 影响程度 |
|------------|------|----------|
| 亚阈值摆幅 | 指数关系 | 中等 |
| 沟道调制 | Vds影响Vth | 较小 |
| 迁移率衰减 | 高Vgs时μ降低 | 较大 |

### 非理想效应

| 效应 | 表达式 | 补偿难度 |
|------|--------|----------|
| 温度漂移 | μ(T) = μ₀·(T/T₀)^(-m) | 中等 |
| 噪声 | I_n = √(4kTγgΔf) | 较易 |
| 失配 | σ(Vth) = A_Vth/√(WL) | 较难 |

## 4.3 线性度测试方法

### 静态线性度测试

**输入斜坡法**：
```
输入: X = 0 → Xmax (线性增加)
输出: 测量Y

理想输出: Y_ideal = X · W
线性度误差: E = |Y - Y_ideal| / Y_FSR
```

### 权重线性度测试

**权重扫描法**：
```
固定输入: X = X_const
扫描权重: W = -1 → +1
测量输出: Y(W)

理想关系: Y ∝ W
非线性误差: E = |Y - k·W| / Y_FSR
```

## 4.4 线性区工作点优化

### 推荐工作点

| 参数 | 优化值 | 理由 |
|------|--------|------|
| Vgs | 0.5V - 1.5V | 避开亚阈值和高Vgs |
| Vds | 0.1V - 0.3V | 保持线性区 |
| Vth | 0.4V - 1.0V | 足够调制范围 |

### 仿真结果

```
工作点: Vgs = 0.9V, Vds = 0.2V
相关系数R²: 0.9987
最大误差: 0.78%
DNL: 0.45 LSB
INL: 0.92 LSB
```

## 4.5 线性度补偿方法

### 方法一：预失真(Pre-distortion)

**原理**：在输入端引入与非线性相反的预失真：

```
X_pre = f^(-1)(X)
Y_actual = f(X_pre) ≈ X
```

**实现**：
- LUT查找表
- 数字预失真
- 模拟预失真电路

### 方法二：分段线性近似

**原理**：将非线性区域分段，每段用线性函数近似：

```
Region 1: X ∈ [0, X1], Y = k1·X
Region 2: X ∈ [X1, X2], Y = k2·X + b2
...
```

### 方法三：闭环校准

**原理**：实时反馈调节：

```
Y_measured → 比较器 → 误差信号 → DAC校准
```

## 4.6 补偿电路设计

### DAC预失真电路

```
输入X → ADC → LUT查找表 → 预失真值X' → DAC → 模拟电压
```

### 校准触发条件

| 触发条件 | 校准方式 |
|----------|----------|
| 上电初始化 | 全LUT校准 |
| 温度变化>10°C | 增量校准 |
| 周期性(1000 cycles) | 局部校准 |

### 补偿效果

| 指标 | 补偿前 | 补偿后 |
|------|--------|--------|
| R² | 0.987 | 0.998 |
| 最大误差 | 1.8% | 0.5% |
| DNL | 0.9 LSB | 0.3 LSB |
| INL | 1.8 LSB | 0.6 LSB |

## 4.7 线性度评估结果

### 仿真配置

```
阵列规模: 128×128
权重精度: 6-bit
输入精度: 4-bit
工作点: Vds = 0.1V
温度: 300K
k值 (mu*Cox*W/L): 1.73e-04 A/V²
```

其中k值由物理参数计算得出：k = μ × Cox × W / L，其中μ=140 cm²/Vs（电子迁移率），Cox=εr×ε0/tox（栅氧电容），W=100nm，L=28nm。

### 线性度指标（仿真结果）

| 测试条件 | R | R² | 最大误差 |
|----------|-----|-----|----------|
| 理想（无失配无噪声） | 1.0000 | 1.0000 | 0.20% |
| 含失配+噪声 | 1.0000 | 1.0000 | 0.20% |

仿真采用物理FET模型计算理想输出（无失配、无噪声的nominal电流），与实际输出（含5%失配和噪声）进行对比。128×128阵列的统计平均效应使得失配和噪声的影响被有效抑制。

---

*本章小结：本章分析了FeFET器件非线性效应的来源，介绍了三种线性度补偿方法（预失真、分段线性、闭环校准），并给出了基于物理FET模型的仿真验证结果。在128×128阵列规模下，统计平均效应使得系统线性度达到R²>0.999。*
