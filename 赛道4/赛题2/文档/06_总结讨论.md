# 第六章 总结与讨论

## 6.1 设计成果总结

### 器件选型

本设计选用**FeFET（铁电场效应晶体管）**作为存储器件，其优势包括：

| 优势 | 描述 |
|------|------|
| 非易失性 | 掉电数据保持，无需刷新 |
| 多比特存储 | 通过极化程度实现 |
| 低功耗 | 亚阈值工作能力强 |
| 与CMOS兼容 | 可集成在标准工艺 |

### 阵列架构

| 参数 | 规格 |
|------|------|
| 阵列规模 | 128×128 |
| 单元结构 | 1T1R |
| 权重精度 | 6-bit |
| 输入精度 | 4-bit |
| 输出精度 | 8-bit |

### 性能指标

| 指标 | 数值 |
|------|------|
| 线性度(R²) | >0.999 |
| 最大误差 | 0.20% |
| 等效精度(ENOB) | 7.85-bit |
| 温度范围 | 250K - 400K |
| 失配容忍度 | σ/μ < 20% (CV<1%) |
| Monte Carlo Yield | 100% (5%容差) |

## 6.2 核心创新点

### 1. 差分对正负信号处理

采用差分对电路统一处理正负权重和输入信号，简化电路设计。

### 2. 工作区间优化

在线性区（Vds < 0.3V）工作，结合预失真补偿，实现高线性度。

### 3. 多层级精度管理

| 层级 | 精度 | 方案 |
|------|------|------|
| 输入 | 4-bit | DAC量化 |
| 权重 | 6-bit | Vth调制 |
| 输出 | 8-bit | ADC采样 |

## 6.3 技术演进路线

```
当前: FeFET + 模拟CIM
  │
  ├── 短期: 优化线性度补偿算法
  │     └── 目标: R² > 0.998
  │
  ├── 中期: 3D堆叠集成
  │     └── 目标: 阵列利用率 > 80%
  │
  └── 长期: 存算一体化SoC
        └── 目标: 完整AI加速器
```

## 6.5 局限性分析

### 技术局限

| 局限 | 影响 | 应对策略 |
|------|------|----------|
| FeFET工艺成熟度 | 器件一致性 | 校准算法 |
| 阵列规模 | 计算精度 | 分块映射 |
| ADC精度 | 输出分辨率 | 逐次逼近型ADC |

### 设计局限

| 局限 | 影响 | 改进方向 |
|------|------|----------|
| 工作电压窗口 | 线性范围受限 | 多电源域设计 |
| 温度敏感性 | 精度漂移 | 温度补偿电路 |

---

*本章小结：本设计完成了基于FeFET的模拟存内计算系统设计，涵盖器件特性分析、工作区间选择、正负信号处理、线性度补偿和精度评估。基于物理FET模型的仿真结果表明，系统在6-bit权重精度下线性度R²>0.999，最大误差0.20%。温度分析（250K-400K）和Monte Carlo失配分析（n=100）验证了系统的鲁棒性，128×128阵列的统计平均效应有效抑制了非理想效应。*
