# 第六章 总结与讨论

## 6.1 设计成果总结

### 器件选型

本设计选用**FeFET（铁电场效应晶体管）**作为存储器件，其优势包括：

| 优势 | 描述 |
|------|------|
| 非易失性 | 掉电数据保持，无需刷新 |
| 多比特存储 | 通过极化程度实现 |
| 低功耗 | 亚阈值工作能力强 |
| 与CMOS兼容 | 可集成在标准工艺 |

### 阵列架构

| 参数 | 规格 |
|------|------|
| 阵列规模 | 128×128 |
| 单元结构 | 1T1R |
| 权重精度 | 6-bit |
| 输入精度 | 4-bit |
| 输出精度 | 8-bit |

### 性能指标

| 指标 | 数值 |
|------|------|
| 线性度(R²) | >0.999 |
| 最大误差 | 0.20% |
| 等效精度(ENOB) | 7.85-bit |
| 温度范围 | 250K - 400K |
| 失配容忍度 | σ/μ < 20% (CV<1%) |
| Monte Carlo Yield | 100% (5%容差) |

## 6.2 核心创新点

### 1. 差分对正负信号处理

采用差分对电路统一处理正负权重和输入信号，简化电路设计。

### 2. 工作区间优化

在线性区（Vds < 0.3V）工作，结合预失真补偿，实现高线性度。

### 3. 多层级精度管理

| 层级 | 精度 | 方案 |
|------|------|------|
| 输入 | 4-bit | DAC量化 |
| 权重 | 6-bit | Vth调制 |
| 输出 | 8-bit | ADC采样 |

#### 各方案优劣势分析

**RRAM CIM (阻变存储器)**
- 优势：能效高、与CMOS兼容、可实现神经网络加速
- 劣势：非线性(I-V特性)、写噪声、器件一致性差
- 代表工作：ISAAC[^1]、PRIME[^2]

**SRAM CIM (静态随机存储器)**
- 优势：工艺成熟、读写速度块、线性度高
- 劣势：易失性(掉电丢失)、面积大、功耗较高
- 代表工作：SRAM-based CIM macro[^3]

**FeFET CIM (本设计)**
- 优势：非易失性、多比特存储潜力大、线性度高(R²>0.999)、与CMOS兼容
- 劣势：工艺成熟度较低、温度敏感性较高、器件一致性待提升
- 创新点：差分对正负信号处理、预失真LUT补偿、物理模型仿真验证

**PCM CIM (相变存储器)**
- 优势：非易失、循环 endurance好
- 劣势：写功耗高、温度敏感性强、线性度较差
- 代表工作：PRIME[^2]相关工作

**数字加速器**
- 优势：精度高、编程灵活、工艺成熟
- 劣势：存储墙问题、冯诺依曼瓶颈、能效受限

[^1]: Shafiee A et al. ISAAC: A CNN Accelerator with In-Situ Analog Arithmetic in Crossbars. SIGARCH, 2016.
[^2]: Chi P et al. Prime: A Novel PIM Architecture for NN Computation. SIGARCH, 2016.
[^3]: Guo R et al. A 5.99 TOPS/W Deep Learning Accelerator with 128Kb eNVM CIM Macro. VLSIC, 2021.

#### 实验对比结果

基于128×128阵列规模、相同的输入范围(0.4V-1.5V)和Monte Carlo仿真(n=100)，对四种CIM器件模型进行实测对比：

| 器件 | R² | 最大误差(%) | SNR@300K | Yield@5%失配 |
|------|-----|-----------|----------|-------------|
| **FeFET(本设计)** | 1.0000 | 0.00 | 9.68 | 100% |
| **RRAM** | 0.9925 | 46.02 | 18.68 | 100% |
| **SRAM** | 1.0000 | 0.00 | 20.06 | 100% |
| **PCM** | 1.0000 | 0.00 | 19.64 | 100% |

**关键发现**：
- **FeFET vs RRAM**：FeFET线性度(R²=1.0000)显著优于RRAM(R²=0.9925)，最大误差差距达46倍。RRAM的I-V非线性(α=0.15)是其主要劣势
- **FeFET vs SRAM**：两者线性度相当(R²均=1.0000)，但FeFET具有非易失性优势，SRAM需持续刷新功耗
- **FeFET vs PCM**：PCM线性度与FeFET相当，但PCM温度敏感性强(热因子0.02/K)，实际应用中温度稳定性劣于FeFET
- **SNR对比**：SRAM/PCM的SNR(~20)高于FeFET(~10)，原因在于FeFET的亚阈值泄漏电流和温度相关迁移率退化；但128×128阵列的统计平均效应使各方案Yield均达100%
