### 第4页: 工作区间分析
```
标题: 工作区间选择的分析

内容:
• VGS-VDS工作区域划分
  - 亚阈值区: VGS < Vth (低功耗但非线性)
  - 线性区: VDS < VGS - Vth (适合模拟计算)
  - 饱和区: VDS > VGS - Vth (放大器应用)

• 最佳工作点选择
  - 选择线性区中段: VGS ∈ [0.4V, 1.6V]
  - VDS = 0.1V (保持线性)
  - 确保所有器件工作在饱和区

• 设计考量
  - 线性度: 线性区最优
  - 功耗: 亚阈值区最低
  - 噪声容限: 饱和区最大

配图: I-V特性曲线示意图
```

### 第5页: 正负信号处理方案
```
标题: 正负信号处理方案

内容:
• 挑战
  - 权重值范围: [-1, +1]
  - 输入电压范围: [0, VDD]
  - 需要双向信号处理

• 差分对电路设计
  - 正信号通路: VGS > Vref
  - 负信号通路: VGS < Vref
  - 参考电压: Vref = 0.9V

• 实现方案
  - 两组交叉阵列
  - 电流差分放大
  - 结果 = I_pos - I_neg

配图: 差分对电路示意图
```
