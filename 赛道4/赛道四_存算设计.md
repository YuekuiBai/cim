## 共同技术背景

### 存算一体架构简介

存内计算技术直接在数据存储的区域进行计算，而非传统计算结构下频繁在存储单元和计算单元间进行数据传递。存内计算技术尝试解决冯·诺伊曼结构的内存速率瓶颈问题，降低通讯开销，提升特定使用场景下的计算效率。

模拟存内计算利用各类传统或新型存储器件进行计算，依托于存储单元的阵列结构及计算特性，模拟存内计算非常适合基于大规模阵列计算的众多 AI 应用。相较于传统数字 AI 加速器，模拟存内计算有着极佳的能效优势。

### 常用存储器件类型

| 器件类型 | 特点 | 适用场景 |
|---------|------|---------|
| **SRAM** | 高速、成熟工艺、与 CMOS 兼容 | 高性能推理 |
| **DRAM** | 高密度、低成本 | 大规模部署 |
| **RRAM** | 非易失、低功耗 | 边缘计算 |
| **MRAM** | 非易失、快速读写 | 高可靠性场景 |
| **FeFET** | 非易失、铁电特性 | 低功耗 AI |
| **Flash** | 高密度、低成本 | 大规模存储 |

## 方向一技术详解：神经网络映射

### 1. 关键技术问题

**问题 1：神经网络层类型适配**

| 层类型 | 映射策略 | 挑战 |
|-------|---------|------|
| **全连接层（FC）** | 直接映射到存算阵列 | 权重大小与阵列规模匹配 |
| **卷积层（Conv）** | im2col + 存算映射 / 直接 2D 映射 | 数据复用、边界处理 |
| **批归一化（BN）** | 融合到相邻层 / 数字后处理 | 参数合并、精度保持 |
| **激活函数** | 查找表（LUT）/ 数字实现 | 非线性拟合、面积开销 |
| **池化层（Pooling）** | 数字逻辑实现 | 控制逻辑复杂度 |

**问题 2：精度与量化**

| 精度方案 | 优点 | 缺点 | 适用场景 |
|---------|------|------|---------| |
| **二进制（1-bit）** | 面积最小、能效最高 | 精度损失大 | 简单分类任务 |
| **低比特（2-4bit）** | 面积与精度平衡 | 需要量化感知训练 | 边缘推理 |
| **中精度（8-bit）** | 精度接近 FP32 | 面积开销较大 | 通用推理 |

**问题 3：非理想因素建模**

| 非理想因素 | 物理来源 | 建模方法 | 影响 |
|-----------|---------|---------|------|
| **器件变异** | 制造工艺波动 | 高斯分布/对数正态分布 | 计算误差 |
| **导线电阻** | 互连寄生效应 | RC 网络模型 | 电压降、延迟 |
| **ADC/DAC 噪声** | 量化误差、热噪声 | 均匀分布/高斯分布 | 精度下降 |
| **温度漂移** | 环境温度变化 | 温度系数模型 | 参数漂移 |

### 3. 建模与仿真流程

```python
# 伪代码示例：存算系统仿真流程

# 步骤 1：器件级建模
def device_model(V, G, non_idealities):
    """
    模拟存算单元 I-V 特性
    V: 输入电压
    G: 电导（权重）
    non_idealities: 非理想因素
    """
    I_ideal = V * G
    I_noise = add_device_noise(I_ideal, non_idealities)
    return I_noise

# 步骤 2：阵列级仿真
def array_simulation(input_vector, weight_matrix, array_config):
    """
    存算阵列矩阵 - 向量乘法
    """
    # 输入数字→模拟转换
    analog_inputs = DAC(input_vector)

    # 并行存算
    analog_outputs = []
    for row in weight_matrix:
        row_current = 0
        for v, g in zip(analog_inputs, row):
            row_current += device_model(v, g, config.non_idealities)
        analog_outputs.append(row_current)

    # 输出模拟→数字转换
    digital_outputs = ADC(analog_outputs)
    return digital_outputs

# 步骤 3：神经网络推理
def neural_network_inference(model, input_data, cim_config):
    """
    在存算系统上运行神经网络
    """
    x = input_data
    for layer in model.layers:
        if is_mapped_to_cim(layer):
            x = array_simulation(x, layer.weights, cim_config)
            x = apply_activation(x, layer.activation)
        else:
            x = layer.forward(x)
    return x

# 步骤 4：性能评估
def evaluate_performance(model, test_dataset, cim_config):
    """
    评估存算系统的精度和能效
    """
    # 精度评估
    correct = 0
    for x, y_true in test_dataset:
        y_pred = neural_network_inference(model, x, cim_config)
        correct += (argmax(y_pred) == y_true)
    accuracy = correct / len(test_dataset)

    # 能效评估
    energy = estimate_energy(model, cim_config)
    latency = estimate_latency(model, cim_config)

    return {
        'accuracy': accuracy,
        'energy': energy,
        'latency': latency,
        'efficiency': accuracy / energy
    }
```

## 方向二技术详解：FET 类器件设计

### 1. FET 类器件简介

在模拟存内计算所使用的各类成熟/新型存储器件中，基于场效应晶体管（Field Effect Transistor, FET）类的器件在部分模拟存内计算场合具有一定的优势。

**常见 FET 类器件：**

| 器件 | 特点 | 应用 |
|------|------|------|
| **FeFET** | 非易失、铁电存储 | 低功耗 AI 推理 |
| **NOR Flash** | 高密度、成熟工艺 | 嵌入式 AI |
| **NAND Flash** | 极高密度、3D 堆叠 | 大规模部署 |
| **P/NMOS** | 标准 CMOS 工艺 | 数字存算混合 |

### 2. 器件物理原理

**FET 类器件工作原理：**

```
        控制栅（CG）
            │
    ┌───────┴───────┐
    │   栅氧化层     │
    ├───────────────┤
    │   沟道（Channel）
    └───────┬───────┘
            │
        源极（S）  漏极（D）

电导调控机制：
- 栅压 Vg 控制沟道电导
- Vg 增大 → 沟道导通 → 电导增大
- 存储电荷可改变阈值电压 Vth
```

**电导与计算的关系：**

| 物理量 | 计算意义 |
|-------|---------|
| 输入电压 V | 网络输入值 |
| 器件电导 G | 网络权重值 |
| 输出电流 I = V × G | 乘加计算结果 |

### 3. 多比特正负信号处理

**挑战：** FET 类模拟存算系统需要处理多比特的正负输入及正负权重信息。

**解决方案：**

| 信号类型 | 处理方法 |
|---------|---------|
| **正负输入** | 双极性编码（差分信号）/ 偏置电压 |
| **正负权重** | 差分对（正权重-负权重）/ 电导调制度 |
| **正负输出** | 电流相减 / 差分放大 |

### 4. 线性度与补偿方法

**非线性来源：**
- 器件 I-V 特性非线性
- 导线电阻/电容寄生效应
- ADC/DAC 量化误差

**补偿方法：**

| 方法 | 原理 | 优缺点 |
|------|------|--------|
| **工作区间选择** | 选择近似线性区域工作 | 简单但牺牲动态范围 |
| **电路级补偿** | 反馈线性化、预失真 | 效果好但电路复杂 |
| **数字补偿** | 查表校正、数字后处理 | 灵活、精度高 |
