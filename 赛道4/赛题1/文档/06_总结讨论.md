# 第六章 总结与讨论

## 6.1 设计总结

### 6.1.1 方案概览

本设计针对ResNet-18神经网络，提出了基于RRAM的模拟存内计算完整映射方案：

| 设计环节 | 关键决策 |
|---------|---------|
| 器件选型 | RRAM (128×128阵列, 8-bit精度) |
| 网络选择 | ResNet-18 (适配CIFAR-10) |
| 映射策略 | im2col + 分块映射 |
| 外围电路 | 8-bit DAC + 10-bit ADC |

### 6.1.2 主要成果

1. **器件特性分析**: 详细分析了RRAM的工作原理、阻变机制和存算阵列实现方式
2. **阵列规格设计**: 确定了128×128阵列规模和外围电路规格
3. **映射方案提出**: 实现了卷积层/全连接层的完整映射方案
4. **性能评估完成**: 系统达到52.4 TOPS吞吐, 90.9 TOPS/W能效

## 6.2 优势分析

### 6.2.1 存算一体优势

| 优势 | 具体表现 |
|------|---------|
| 能效提升 | 30倍以上能效增益 (vs 数字加速器) |
| 延迟降低 | 减少数据搬移，缩短关键路径 |
| 面积优化 | 存储与计算融合，节省硅面积 |
| 扩展性好 | 阵列可堆叠，便于规模化 |

### 6.2.2 模拟计算优势

- **并行度高**: 阵列级并行矩阵运算
- **功耗低**: 模拟乘法无需频繁数据访问
- **可扩展**: 易于与先进制程集成

## 6.3 挑战与应对

### 6.3.1 主要挑战

| 挑战 | 描述 | 影响 |
|------|------|------|
| 非线性误差 | RRAM I-V特性非线性 | 计算精度下降 |
| 器件差异 | 工艺波动导致单元不一致 | 权重精度损失 |
| 阵列利用率 | 分块映射导致利用不均 | 资源浪费 |
| ADC精度 | 10-bit限制整体精度 | 量化误差 |

### 6.3.2 应对方案

1. **非线性补偿**:
   - 电路级: 预充电补偿
   - 算法级: 数字校准

2. **器件差异缓解**:
   - 定期校准
   - 冗余设计
   - 差分结构

3. **利用率优化**:
   - 层间复用阵列
   - 动态分块策略

## 6.5 结论

本设计完成了从器件选型到性能评估的完整存内计算方案，验证了RRAM存算一体架构在神经网络推理中的可行性。系统在高能效和低延迟方面展现出显著优势，同时面临精度和稳定性的挑战。

**核心指标汇总**:

| 指标 | 数值 | 对比数字加速器 |
|------|------|---------------|
| 吞吐量 | 52.4 TOPS | 相当 |
| 能效 | 90.9 TOPS/W | 30x提升 |
| 精度损失 | ~6% | 可接受 |
| 面积 | 10.49 mm² | 更小 |

随着器件和电路技术的进步，存内计算有望在AI芯片领域发挥越来越重要的作用。
