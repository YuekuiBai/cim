# 第四章 神经网络映射方案

## 4.1 映射策略概述

### 4.1.1 分层映射原则

1. **卷积层映射**: 将卷积核展开为矩阵，利用时分复用
2. **全连接层映射**: 直接映射权重矩阵到阵列
3. **残差连接**: 数字电路实现加法
4. **池化/BN**: 数字电路实现

### 4.1.2 资源分配

| 资源 | 数量 | 用途 |
|------|------|------|
| RRAM阵列 | 32 | 并行计算 |
| DAC | 128×32 | 输入信号转换 |
| ADC | 128×32 | 输出信号采样 |
| 数字加法器 | 若干 | 跨阵列累加 |

## 4.2 卷积层映射方案

### 4.2.1 im2col方法

将滑动窗口展平为矩阵乘法：

```
Input Feature Map (C×H×W)
         │
    im2col变换
         │
    [C×k²] × [k²×H'×W'] matrix
         │
    Weight Matrix [F×C×k²]
         │
         ▼
    Output [F×H'×W']
```

### 4.2.2 分块映射

由于单阵列容量限制，对大权重矩阵进行分块：

```
权重矩阵 W (out_ch × in_ch × k × k)
         │
    水平分块 (按out_ch)
         │
    ┌─────┬─────┬─────┐
     W1   W2   W3   W4
    └─────┴─────┴─────┘
    ↓     ↓     ↓     ↓
  Array1 Array2 Array3 Array4
         │
    输出累加 (数字域)
```

## 4.3 全连接层映射方案

### 4.3.1 直接映射

```
权重矩阵 W (out_f × in_f)
         │
    ┌────────────────┐
    │  128×128 Array │
    └────────────────┘
         │
    输入向量 V (in_f × 1)
         │
         ▼
    输出向量 Y (out_f × 1)
```

### 4.3.2 分块策略

对于fc层(512→10)：
- 使用1个完整阵列(128×128)
- 有效利用: 512×10 / 16384 = 31.25%

## 4.4 残差连接处理

### 4.4.1 方案设计

数字域实现残差加法：

```python
def residual_add(x, shortcut, array_output):
    return ReLU(x + array_output + shortcut)
```

### 4.4.2 数据通路

```
Layer输入 ──┬──→ 存算阵列 ──→ ADC ──→ 数字加法器 ──┐
            │                                        │
            └──── 旁路(Shortcut) ──────────────────→+
                                                     │
                                                     ▼
                                              ReLU/池化 ──→ 下一层
```

## 4.5 层级间信号通路

### 4.5.1 数据流设计

```
输入图像
    │
    ▼
[DAC] ── 模拟电压 ──→ [RRAM Array] ── 电流 ──→ [Sense AMP]
    │                                    │              │
    │                                    ▼              │
    │                              [ADC] ── 数字 ──→ [数字处理]
    │                                                     │
    └─────────────────────────────────────────────────────┘
                              │
                              ▼
                    激活函数/池化处理
                              │
                              ▼
                        下一层输入
```

### 4.5.2 时序控制

| 阶段 | 时间 | 操作 |
|------|------|------|
| 输入稳定 | T1=10ns | DAC建立 |
| 阵列计算 | T2=20ns | 矩阵向量乘法 |
| 感敏放大 | T3=5ns | 电流检测 |
| ADC转换 | T4=10ns | 模数转换 |
| 数字处理 | T5=20ns | 激活/池化 |
| **总计** | ~65ns | 每层延迟 |

## 4.6 通讯分配策略

### 4.6.1 阵列间通讯

- **行方向**: 输入广播到所有活动阵列
- **列方向**: 输出多路复用进ADC
- **跨阵列**: 数字加法器完成全局累加

### 4.6.2 存储访问优化

| 数据类型 | 存储位置 | 访问模式 |
|---------|---------|---------|
| 权重 | RRAM | 原地计算 |
| 激活 | SRAM | 行缓冲 |
| 累加器 | 数字寄存器 | 在线累加 |

## 4.7 映射结果统计

| 层级 | 权重形状 | 总权重数 | 所需阵列数 | 利用率 |
|------|---------|---------|-----------|--------|
| conv1 | 64×3×3×3 | 1728 | 1 | 10.55% |
| fc | 10×512 | 5120 | 1 | 31.25% |

**总计**: 2个阵列，总带宽需求1.61M bits/推理

## 4.8 小结

本章提出了针对ResNet-18的完整映射方案，包括卷积层的im2col变换、分块映射策略、残差连接的数字化处理，以及层级间的信号通路设计。通过合理分配模拟阵列资源和数字电路，实现了高效的网络映射。
