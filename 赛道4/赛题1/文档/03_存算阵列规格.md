# 第三章 存内计算阵列规格信息

## 3.1 阵列架构设计

### 3.1.1 阵列规格

| 参数 | 规格值 | 说明 |
|------|--------|------|
| 阵列规模 | 128×128 | 行×列 |
| 单元类型 | 1T1R | 一晶体管一电阻 |
| 存储密度 | 4T+1R/bit | 含选择晶体管 |
| 电导范围 | 10μS ~ 1mS | 可编程范围 |
| 位精度 | 8-bit | 权重/输入精度 |

### 3.1.2 单元电路

```
    WL (Word Line)
      │
   ┌──┴──┐
   │     │
  Gate  Source
   │     │
   ├─────┼──── R_RAM ── BL (Bit Line)
   │     │          │
   └─────┘          │
   NMOS             │
   Select           │

1T1R结构：NMOS作为选择管，控制RRAM单元的读写访问
```

## 3.3 精度分析

### 3.3.1 输入端精度

- **DAC量化误差**: ±1/2 LSB = ±1.95 mV (1V/512)
- **输入噪声**: < 1 mV rms
- **输入线性度**: INL < 0.5 LSB

### 3.3.2 权重端精度

- **权重量化**: 8-bit (256 levels)
- **权重编程精度**: ±2 LSB
- **权重保持特性**: ±1% (10年)

### 3.3.3 输出端精度

- **ADC量化误差**: ±1/2 LSB = ±0.98 mV
- **输出噪声**: < 0.5 mV rms
- **感敏放大器精度**: 8-bit effective

### 3.3.4 端到端精度

综合各模块贡献：
- **理论精度**: ~7.5-bit effective
- **主要限制**: 非线性误差 > 量化误差

## 3.4 功耗分析

| 模块 | 功耗 |
|------|------|
| RRAM阵列(128×128) | ~1.55 W (动态) |
| DAC(128通道) | 256 mW |
| ADC(128通道) | 640 mW |
| 数字控制逻辑 | 50 mW |
| **总计** | ~2.5 W |

| 模块 | 面积 |

## 3.6 小结

本章详细描述了128×128规模的RRAM存算阵列及其外围电路规格。8-bit DAC/10-bit ADC的配置配合128×128阵列，能够满足ResNet-18的矩阵运算需求，同时兼顾能效和精度。
