# 第一章 存储器件的工作原理及特性介绍

## 1.1 器件选型

**选定器件**：RRAM (阻变存储器，Resistive Random Access Memory)

**选型理由**：
- 非易失性存储，无需持续供电维持数据
- 与标准CMOS工艺兼容，易于三维集成
- 读写速度适中（ns级），适合存算一体场景
- 器件特性支持模拟矩阵运算

## 1.2 RRAM工作原理

### 1.2.1 基本结构

RRAM器件采用"金属-绝缘层-金属"(MIM)三明治结构：
- 顶电极(Top Electrode): TiN/Ta
- 绝缘层(Resistance Switching Layer): HfO₂/Ta₂O₅
- 底电极(Bottom Electrode): Pt/TiN

### 1.2.2 阻变机制

**SET过程（高阻→低阻）**：
- 施加正向电压
- 氧离子向底电极迁移
- 形成导电细丝(CF)
- 器件电阻降低

**RESET过程（低阻→高阻）**：
- 施加反向电压
- 导电细丝断裂
- 器件电阻升高

### 1.2.3 电学特性

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 导通电阻 | 1kΩ ~ 1MΩ | 可通过编程调节 |
| 导纳范围 | 10μS ~ 1mS | 对应Gmin ~ Gmax |
| 保持时间 | >10年 | 非易失性 |
| 写入速度 | ~100ns | SET/RESET时间 |
| 耐擦写次数 | >10⁶ cycles | 可靠性 |
| 功耗 | < 0.1pJ/bit | 低能耗 |

## 1.3 存算阵列实现原理

### 1.3.1 交叉开关阵列结构

```
        Word Line (WL)
            │
    ┌───────┼───────┐
    │       │       │
   BL0      BL1     BL2    Bit Line (BL)
    │       │       │
    ├───────┼───────┤
    │  RRAM │  RRAM │  ──► 存储权重值
    │  Cell │  Cell │
    ├───────┼───────┤
    │  RRAM │  RRAM │
    │  Cell │  Cell │
    └───────┴───────┘
```

### 1.3.2 矩阵-向量乘法实现

**数学原理**：
$$y_j = \sum_i G_{ij} \cdot V_i$$

其中 $G_{ij}$ 为第(i,j)个RRAM单元的电导，$V_i$ 为输入电压，$y_j$ 为输出电流。

**工作流程**：
1. 将权重矩阵编码为RRAM阵列的电导分布
2. 输入向量通过DAC转换为模拟电压信号
3. 各行电压施加，列端感知的电流求和即为矩阵乘法结果
4. ADC将模拟电流转换为数字输出

### 1.3.3 仿真参数配置

本设计采用的阵列规格：

| 参数 | 数值 |
|------|------|
| 阵列规模 | 128×128 |
| 电导范围 | 10μS ~ 1mS |
| 位精度 | 8-bit |
| 工作电压 | 0.5V |
| 温度 | 300K |

## 1.4 非理想因素分析

### 1.4.1 非线性特性

RRAM器件的I-V特性存在非线性，本设计采用三次多项式建模：

$$V_{out} = \alpha \cdot V_{in}^3 + (1-\alpha) \cdot V_{in}$$

其中α为非线性因子，典型值为0.05。

### 1.4.2 噪声特性

| 噪声源 | 标准差 | 影响 |
|--------|--------|------|
| 热噪声 | 0.01 | SNR约25dB |

### 1.4.3 仿真结果

基于128×128交叉阵列的仿真结果：

| 指标 | 数值 |
|------|------|
| 最大输出电流 | 50.76 mA |
| 最小输出电流 | 1.94 mA |
| 平均功耗 | 1.55 W |
| 阵列效率 | 94.6 nW/cell |
| 信噪比(SNR) | 7.09 dB (无噪声) / 25.6 dB (线性模型) |

## 1.5 小结

RRAM器件以其非易失性、低功耗、与CMOS兼容的特性，成为模拟存内计算的理想选择。其交叉开关阵列结构能够高效实现矩阵-向量乘法，为神经网络推理提供硬件基础。

仿真结果表明，在128×128阵列规模下，系统可实现约1.55W的平均功耗和7-25dB的信噪比，非线性因素对精度有一定影响但可通过校准技术改善。
